# Priority-Based Loan Approval System

A Java-based desktop application that automates loan evaluation and approval using a rule-based scoring system, enabling consistent, efficient, and structured decision-making.

---

## Project Overview

- **Goal:** Automate and standardize the loan approval process  
- **Focus:** Decision-making logic, role-based access, and system workflow  
- **Output:** Functional Java application with GUI, scoring system, and data persistence  

---

## Technologies Used

- Java  
- Java Swing  
- File Handling / Database  
- NetBeans  

---

## Problem

Manual loan evaluation is time-consuming, inconsistent, and prone to subjective decision-making without a structured framework.

---

## Solution

The system applies a rule-based scoring mechanism to evaluate applicants based on defined criteria, ensuring faster, fairer, and more consistent loan approval decisions.

---

## Tools Used

- **NetBeans IDE** – Used to develop, run, and manage the Java application  
- **Java Swing** – Built the graphical user interface including forms, dashboards, and navigation  
- **File Handling / Database** – Stored applicant and loan data for retrieval and updates  

---

## Key Features

- Role-based system (Applicant, Loan Officer, Admin)  
- Loan application submission interface  
- Automated scoring and prioritization system  
- Approval and rejection decision support  
- Data storage and retrieval  
- Organized tab-based interface
